## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

If you discover a security vulnerability, please report it privately to ensure the safety of all users during our beta.

### How to Report
Send an email to: **usernilang@gmail.com**

Please include the following in your report:
1. **Description:** A brief summary of the vulnerability.
2. **Steps to Reproduce:** A clear set of steps (or a Proof of Concept script) to demonstrate the issue.

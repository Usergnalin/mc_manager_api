# Minecraft Manager API (MC Manager API)

A API designed to manage a fleet of Minecraft servers through remote agents. This platform enables centralized control, real-time monitoring, and secure team collaboration for Minecraft server infrastructure.

We welcome all security improvements and bug fixes; please review our CONTRIBUTING.md for our development process and licensing terms for contributions.

This project is in Beta. We take security seriously. If you find a vulnerability, please do not open a public issue. See SECURITY.md for details.
